# 关于我们
BMInf由OpenBMB（Open Lab for Big Model Base）开发与维护。OpenBMB由北京智源人工智能研究院（BAAI）与清华大学共同支持成立。

OpenBMB旨在打造大规模预训练语言模型库与相关工具，加速百亿级以上大模型的训练、微调与推理，降低大模型使用门槛。在此基础上依托开源社区进一步推动预训练语言模型相关生态发展，打造智能时代的AI基础设施与应用范式。
